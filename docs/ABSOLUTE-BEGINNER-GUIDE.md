## 📥 Installation Guide (5 Minutes)

Let's build your website!

Follow these steps one at a time.

### Step 1: Get NodePress

You need to download NodePress to your computer.

**Option A: Download the ZIP file (EASIEST)**

1. Open your web browser (Chrome, Edge, or Safari)
2. Go to this address: `https://github.com/superdevids/nodepress`
3. Look for a green button that says **"Code"**
4. Click it
5. Click **"Download ZIP"**
6. Find the downloaded ZIP file in your "Downloads" folder
7. RIGHT-CLICK the ZIP file
8. Click **"Extract All"** (Windows) or double-click to open it (Mac)
9. You should now see a folder called `nodepress`

[SCREENSHOT: GitHub page showing the green "Code" button and "Download ZIP" option]

![Screenshot placeholder: GitHub download page]

**Option B: Open Terminal (for advanced users)**

If you like using the command prompt:

```bash
git clone https://github.com/superdevids/nodepress.git
cd nodepress
```

Don't worry if this looks scary. Just use Option A above!

---

### Step 2: Run the Installer

Now open the `nodepress` folder you just downloaded.

Look for the installer file.

#### 🪟 For Windows Users

1. Find the file called **`install.bat`**
2. DOUBLE-CLICK it
3. A black window will pop up
4. Do NOT close this window
5. Wait while it installs things

[SCREENSHOT: Windows folder showing install.bat file highlighted]

![Screenshot placeholder: Windows folder with install.bat]

#### 🍎 For Mac Users

1. Find the file called **`install.sh`**
2. DOUBLE-CLICK it
3. If you see a message about "Terminal" - click **"Open"**
4. A black window will appear
5. Do NOT close this window
6. Wait while it installs things

[SCREENSHOT: Mac folder showing install.sh file highlighted]

![Screenshot placeholder: Mac folder with install.sh]

#### 🐧 For Linux Users

1. Open Terminal
2. Type: `bash install.sh`
3. Press Enter
4. Wait while it installs things

---

### Step 3: Let the Installer Do Its Magic

This is the cool part.

The installer will check your computer automatically.

Here is what happens:

```
✅ Checking your computer...
   → Do you have Docker? Checking...
   → Do you have Node.js? Checking...
   → Installing what you need...
   → Starting your website...
   → Opening your browser...
```

**What if the installer says I need something?**

No problem! The installer will tell you exactly what to do.

**Case 1: "Please install Node.js"**

1. The installer will show you a website link
2. Click the link
3. Download the file
4. Double-click to install it
5. Run the installer again

**Case 2: "Please install Docker"**

1. The installer will show you a website link
2. Click the link
3. Download Docker Desktop
4. Install it (just double-click and follow the steps)
5. Open Docker Desktop (look for the whale icon 🐳)
6. Wait for Docker to start
7. Run the installer again

**Case 3: "Please install Git"**

1. Go to `https://git-scm.com/downloads`
2. Download for your computer
3. Install it
4. Run the installer again

## ❓ Common Questions

### Q: Is NodePress really free?

**Yes!** 100% free. No hidden costs.

It is "open source" — meaning anyone can use it for free.

### Q: Can I use it for my business?

You can try it out.

But please know: NodePress is still in **beta**.

That means it is still being tested.

For a business website, you might want to wait.

Or ask a developer to help you set it up properly.

### Q: Can I migrate my old site here?

Yes! NodePress has an import tool for that.

It can import your posts, pictures, and users from most CMS platforms.

See the Migration Guide for step-by-step help.

### Q: What if I get stuck?

We are here to help!

- 📖 **Read this guide again** — You might have missed a step
- ❓ **Check the FAQ** — `docs/FAQ.md`
- 💬 **Ask on GitHub Discussions** — `https://github.com/superdevids/nodepress/discussions`
- 🐛 **Report a problem** — `https://github.com/superdevids/nodepress/issues`

### Q: Can I add more features later?

Yes! NodePress has 13 built-in features (called "plugins"):

- SEO (get found on Google)
- Comments (let people talk)
- Analytics (see your visitors)
- Security (stay safe)
- Forms (contact forms)
- Backup (save your work)
- And more!

Go to **Plugins** in your dashboard to turn them on.

### Q: How is NodePress different from other CMS platforms?

| Feature             | NodePress           |
| ------------------- | ------------------- |
| Price               | Free                |
| Ease of use         | Easy                |
| Number of plugins   | 13                  |
| Technology          | JavaScript (modern) |
| Speed               | Very fast           |
| Ready for business? | Almost (in beta)    |

### Q: What does "beta" mean?

Beta means the software is still being tested.

It works! But there might be a few bugs.

The makers are still adding polish.

Think of it like a movie before it hits theaters.

The story is great, but some scenes are still being edited.

### Q: Do I need to know what "localhost" means?

No. Just remember this:

When you see `http://localhost:3000`, it means:

"Your own computer, showing your website."

Only you can see it right now.

To share it with the world, you need to put it on the internet.

That is called "deploying."

For now, just enjoy building your site locally!
